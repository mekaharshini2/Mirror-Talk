Mirror Talk is a simple web-based application that helps users practice real-life conversations in a safe, supportive, and interactive environment. It is designed to help people overcome social anxiety and build communication confidence through AI-guided practice scenarios.

Features

Practice real-life conversations like:

Confessing feelings

Apologizing to a friend

Making new friends

AI-generated responses for guidance

Speech recognition using microphone input

Text-to-speech AI replies

Simple and clean user interface

The homepage introduces the concept of practicing conversations to build confidence.
The session page provides a chat interface where users can type or speak, and the AI responds with guidance.

Technologies Used

HTML

CSS

JavaScript

Web Speech API (Speech Recognition & Text-to-Speech)

How to Run

Download or clone the project.

Open the project folder.

Double-click index.html or open it in a web browser.

Click Get Started to begin practicing conversations.
How It Works

User logs in.

Selects a conversation scenario.

Starts a practice session.

Types or speaks a message.

AI provides supportive responses and coaching.
